#  Exif-DATA script :

##  Prerequisites 

install Pillow and moviepy library.

```
pip install Pillow
```
```
pip install moviepy
```

How to use it

```
git clone https://github.com/daitohck/Exif-Data/
cd Exif-Data
```
Launch the script 
```
python script_metadata.py chemin/vers/votre/image.png
```
or 
```
python script_metadata.py chemin/vers/votre/video.mp4
```
